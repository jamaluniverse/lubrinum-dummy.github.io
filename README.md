# Lubrinum | Perpetual Motion Architecture

Lubrinum is a premium industrial web application focused on sensory engineering and superlubricity. This project implements a modern, high-contrast design system based on official brand guidelines, utilizing a stack of Tailwind CSS and Vue.js.

## 🚀 Quick Start

1.  **Clone the repository**:
    ```bash
    git clone https://your-repo-link/website-lubrinum.git
    cd website-lubrinum
    ```
2.  **Open in Browser**:
    Simply open `index.html` in any modern web browser. No compilation or heavy installation required (utilizes CDN-based delivery for rapid prototyping).

## 🛠 Tech Stack

-   **Structure**: HTML5 (Semantic)
-   **Styling**: [Tailwind CSS v3](https://tailwindcss.com/) (delivered via CDN)
-   **Logic**: [Vue.js 3](https://vuejs.org/) (delivered via CDN)
-   **Typography**: Work Sans (Headings) & Sofia Sans (Body)
-   **Assets**: Optimized SVG graphics

## 🎨 Branding Guideline Implementation

-   **Primary (Performance Red)**: `#D00000`
-   **Secondary (Cobalt Core Blue)**: `#191970`
-   **Text (Vivid Black / Clear White)**: `#020000` / `#F5F9FA`
-   **Typography**: Headings use **Work Sans**, body text uses **Sofia Sans**.
